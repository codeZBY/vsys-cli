# Usage

一、下载并全局安装

    npm install vsys-cli -g

二、初始化项目

    vsys init <name>
    
    
三、初始化页面

    vsys page
    
    1、指定初始化路径时，是从 src/views/app/ 开始的；如：test1/test2,模板会初始化到src/views/app/test1/test2/[name].vue
    2、vsys page 命令需要在项目根目录中执行
    
四、查看所有可执行命令
    
    直接输入vsys或者vsys -l，即可查看所有option和command
